## API

| 参数     | 说明               | 类型        | 默认值 |
| -------- | ------------------ | ----------- | ------ |
| options     | 以配置形式设置子元素     | `string[] \| number[] \| Array<CheckboxOptionType>`    |   -     |
| value     | 用于设置当前选中的值     | `any`    |   -     |
| onChange | 选项变化时的回调函数 | `function(e:Event)` |   -     |
